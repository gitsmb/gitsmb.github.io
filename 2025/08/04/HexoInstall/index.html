Hexo 是一个用 Node.js 开发的快速、简洁的静态博客框架。它让你能用 Markdown 轻松写作，
然后一键生成静态网页文件，这些文件可以非常方便地部署到 GitHub Pages 等托管平台，
构建高性能的个人博客或网站。它适合追求简洁、高效和免费托管的用户。
